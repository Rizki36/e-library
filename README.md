# SiPus Alpha
Sistem Perpustakaan

- Bahasa pemrograman : php
- Framework : Codeigniter 3
- Database : Ms Access

## Menjalankan Aplikasi
- Letakkan folder di htdocs
- Buat database dengan nama sipus 
- Import file database.sql
